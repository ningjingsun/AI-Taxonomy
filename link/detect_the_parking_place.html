AI Parking


AI Parking provides a user a best parking spot and assists parking. Users can reduce time needed for parking, parking fee, and the possibility of accidents while parking. This feature can create a new mapping system, increas the efficiency using space and reduce the human labor for parking.


Industry: Transportation
Service Type: B2C

Sequences

1. Identify the willingness to park  >  2. Detect the parking place  >  3. Compare potential parking place  >  4. Estimate the size of parking space  >  5. Compose the driving path    

Reinforcement Machine Learning

Input Data:


Dataset of parking space images, satellite and GPS data

Output Data:


Label saying whether there is a possible parking place

Data Features:


Availability of parking or not, possible routes to park
